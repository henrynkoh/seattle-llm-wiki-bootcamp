# Tistory — 포스트 초안 (Seattle LLM Wiki Bootcamp)

**제목:** [시애틀 스타트업] 1주 만에 만드는 LLM Wiki — 부트캠프 커리큘럼 공개

**카테고리:** IT·스타트업 / 생산성

---

### 들어가며

자료는 많은데 **검색 가능한 지식**으로 남지 않는 문제, LLM 시대에 더 자주 보입니다. 이 글에서는 **Seattle LLM Wiki Bootcamp**라는 **공개 마이크로사이트**를 소개합니다. 대상은 **시애틀 권역의 소규모 스타트업 코호트**이며, 누구나 일정과 방법을 따라 해 볼 수 있게 구성되어 있습니다.

**링크:** https://seattle-llm-wiki-bootcamp-cs.vercel.app

---

### 사이트 구성

| 경로 | 내용 |
| --- | --- |
| `/` | Claude Code 치트시트 (한국어) |
| `/curriculum` | 7일 일정 |
| `/discovery` | 고객·시장 자료 합성 트랙 |
| `/graphify` | Graphify CLI 트랙 |
| `/logistics` | 사전 준비·운영 |
| `/facilitators` | 퍼실리테이터 런북 |

---

### 핵심 아이디어

- **Raw → Wiki → Query:** 원본을 남기고, 구조화하고, 질문에 답할 수 있게 만든다.  
- **Discovery track:** `/raw`에 흩어진 **messy artifact**를 모아 **테마·인사이트**로 만드는 병렬 주간.  

---

### 맺음말

직접 코호트를 열 계획이 있다면 로지스틱스와 체크리스트를 그대로 복사해 로컬에 맞게 수정하면 됩니다. 모집 공고를 낼 때는 **일정·장소·선발 기준**만 채워 넣으세요.
