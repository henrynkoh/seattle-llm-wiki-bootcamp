## Post body

**What we added**

The **Seattle LLM Wiki Bootcamp** already has a public microsite with the full **7-day curriculum**, **Discovery & Synthesis** track, logistics, and facilitator materials. We’ve also included an **optional** second Next.js app in the repository: **`seattle-llm-wiki-bootcamp-lab`**.

**Why it exists**

Facilitators asked for something that reads like a **workshop manual** on a second screen:  
- One page per day with **ordered steps**  
- Optional **checkboxes** (stored in your browser)  
- **Playbook** pages for Dataview, DataviewJS, Templater, Linter, Style Settings, themes, plugins, mobile, sync, and backup  

**What it does not replace**

Your **Obsidian vault** on disk is still the source of truth. The lab app is a **guide**, not a hosted wiki.

**How to run it**

Clone the GitHub repository, `cd seattle-llm-wiki-bootcamp-lab`, `npm install`, `npm run dev`. You can deploy it separately on Vercel if you want a private URL for cohorts.
