# 네이버 블로그 — 부트캠프 Lab 웹앱(선택)

**제목 예시:** 시애틀 LLM 위키 부트캠프 — 옵시디언 실습용 ‘Lab’ 웹앱 안내

**태그:** 시애틀, 스타트업, 옵시디언, LLM, 부트캠프, 생산성, AI

---

## 리드

공개 사이트에는 **7일 커리큘럼**과 **Discovery & Synthesis** 트랙이 있습니다. 저장소에는 **추가로** Next.js 기반 **Lab 앱**(`seattle-llm-wiki-bootcamp-lab`)이 포함되어 있어, **일별 단계**와 **플레이북**(Dataview, DataviewJS, Templater 등)을 한 화면에서 따라가기 좋습니다.

**공식 사이트:** https://seattle-llm-wiki-bootcamp-cs.vercel.app

---

## 본문 요약

1. **Lab**은 볼트를 대체하지 않습니다. **로컬 옵시디언**이 본체입니다.  
2. **체크리스트**는 브라우저 **localStorage**에만 저장됩니다(기기 간 동기화 없음).  
3. 실행: 저장소 클론 후 `seattle-llm-wiki-bootcamp-lab` 폴더에서 `npm install` → `npm run dev`.  
4. 메인 사이트의 **클라우드 치트시트·커리큘럼**과 함께 쓰면 팀 온보딩이 쉬워집니다.
