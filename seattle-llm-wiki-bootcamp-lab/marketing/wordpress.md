## Gutenberg-friendly sections

### Lead paragraph

The **Seattle LLM Wiki Bootcamp** public site hosts the full **7-day curriculum**, **Discovery & Synthesis** track, logistics, and facilitator runbook. The **GitHub repository** also contains an optional **Lab** web app (`seattle-llm-wiki-bootcamp-lab`) for cohorts who want a **dedicated UI** next to their Obsidian vault.

### What the Lab includes

- Day-by-day pages (`/day/1` … `/day/7`) with ordered steps  
- Optional **browser checklists** (local storage only)  
- **Playbooks** for Dataview, DataviewJS, Templater, Linter, Style Settings, themes, plugins, mobile, sync, and backup  

### Canonical link

[Seattle LLM Wiki Bootcamp](https://seattle-llm-wiki-bootcamp-cs.vercel.app)

### For developers

Clone the repository, open the `seattle-llm-wiki-bootcamp-lab` folder, run `npm install` and `npm run dev`. Deploy as a separate Vercel project if you want a private cohort URL.

### Closing CTA

**Founders:** start with the public curriculum. **Facilitators:** add the Lab to your screen share when you want a **step-by-step** companion.
